How to export this as og.png:
1. Open this file in Chrome at 100% zoom on a 1920×1080+ display.
2. Right-click the dark card above → Inspect.
3. In DevTools, right-click the <div class="og"> element → Capture node screenshot.
4. Chrome saves a 1200×630 PNG. Rename it og.png and drop it in the project root.
5. Redeploy. og.png will now be served at /og.png for all OG/Twitter previews.

Faster alternative: open this file in a headless browser (Puppeteer / Playwright) and screenshot the .og selector. A one-line script if you want to automate it.
